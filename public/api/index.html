和歌山大学非公式シラバス検索API

使い方

/api/select

概要

シラバストップページの選択肢

パラメータ

なし

返り値の例

/api/select

/api/search

概要

授業の検索

パラメータ

GETリクエストのパラメータ一覧。正規表現に対応

キー 説明 値の型
page(必須) ページ数 1以上の整数
title タイトル 整数
folder フォルダー 整数
name 科目名 文字列(正規表現に対応)
editors 担当教員 文字列(正規表現に対応)
overview 授業の概要・ねらい 文字列(正規表現に対応)

返り値の例

/api/search?name=論&page=1
